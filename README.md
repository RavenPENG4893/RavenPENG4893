<summary><h2>🔧 &nbsp;Setup Guide（点击展开配置教程）</h2></summary>

### 📋 第一步：创建同名仓库

1. 在 GitHub 新建仓库，名称必须是 `RavenPENG4893`（和用户名一模一样）
2. 设为 **Public**
3. 勾选 **"Add a README file"**
4. 用本文件内容替换默认 `README.md`

### 🐍 第二步：配置贪吃蛇 GitHub Action

在仓库中创建 `.github/workflows/snake.yml`，内容如下：

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */6 * * *"
  workflow_dispatch:

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    
    steps:
      - name: Generate Snake Game
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: RavenPENG4893
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          
      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

然后：
1. **Settings → Actions → General → Workflow permissions** 改为 **Read and write**
2. **Actions** 标签 → 手动 Run `Generate Snake Animation`
3. 等执行成功，`output` 分支自动创建

### ⚠️ 第三步：替换占位符
